What is this 'signature.asc' you sent me?

This file is a PGP signature. Put simply, if you had certain program installed on your computer, you could use this file to prove that the email came from me, instead of somebody pretending to me. If you're not worried about that and don't want to install anything, it is safe to ignore this file.

You can learn more about how to get started protecting your own mail here. All the software is available for free and developed in the open by the community.
